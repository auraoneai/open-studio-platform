# Open Studio GitHub Discussions Seeds

Create these threads when GitHub Discussions is enabled for the flagship repositories. Keep the safety text intact so contributors do not post secrets, private prompts, traces, datasets, credentials, or unreleased customer data.

## Rubric Studio Open

Repository: `auraoneai/rubric-studio-open`

### Welcome: What are you evaluating?

Share the domain you are building rubrics for, whether you are using the desktop
app, browser edition, CLI, or VS Code extension, and what export format matters
most to you.

### Show your first rubric

Post a small rubric project or screenshot. Avoid sharing private samples,
provider keys, customer data, or unreleased benchmark data.

### Export adapters: what should ship next?

Use this thread for lm-eval-harness, Inspect, OpenAI Evals, Promptfoo, Hugging
Face Hub, Surge SOW, and Scale task-spec feedback.

### Security and privacy questions

Ask about local-first behavior, telemetry, crash reporting, keychain storage,
intake export contents, signing, notarization, and update verification.
