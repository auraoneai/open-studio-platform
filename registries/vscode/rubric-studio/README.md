# Rubric Studio Open for VS Code

Marketplace manifest for `auraone.rubric-studio`.

The extension source opens the same project format as the desktop app and
provides project validation. The checked-in package manifest currently declares
`0.0.1`; the prior `0.1.0` publication claim has not been independently
reverified and must not be used as release evidence.

Before Marketplace publication, build a VSIX from the exact tagged source,
verify the publisher identity, inspect the package contents, install it into a
clean VS Code profile, and record the public Marketplace URL in the canonical
release-evidence manifest.
