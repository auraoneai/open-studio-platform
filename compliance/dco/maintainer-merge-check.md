# Maintainer Merge Check

Before merging a PR to an Open Studio repo:

- [ ] Every commit has a valid `Signed-off-by:` trailer.
- [ ] The author understands that DCO is a contribution attestation, not a CLA.
- [ ] Squash commits preserve at least one accurate sign-off trailer for each contributor.
- [ ] Co-authored commits preserve `Co-authored-by:` and `Signed-off-by:` trailers where appropriate.
- [ ] Security-sensitive changes have the required Platform Security review.
