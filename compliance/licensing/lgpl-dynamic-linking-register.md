# LGPL Dynamic Linking Register

Date opened: 2026-05-13
Owner: Platform Security

Open Studio products ship under MIT. GPL and AGPL dependencies are not permitted
in shipped binaries. LGPL dependencies are allowed only when dynamically linked,
documented here, and approved by Platform Security.
