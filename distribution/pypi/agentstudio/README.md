# agentstudio

`agentstudio` is the Agent Studio Open command-line interface package.

The CLI is reserved for the Agent Studio Open release line and exposes the
same local-first workflows as the desktop app:

- connect to MCP and A2A endpoints
- record and replay agent sessions
- compare model behavior against deterministic tool outputs
- run risk scans and A2A contract checks
- export trace cards, JUnit XML, GitHub Actions, and PR comments

Package publication is blocked until the CLI binary and Python package are
built from the Agent Studio Open source repository and signed release artifacts
are available.
