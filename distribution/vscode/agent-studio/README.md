# Agent Studio Open for VS Code

The VS Code extension package reserves the Marketplace identity
`auraone.agent-studio`.

Release scope:

- discover MCP server manifests from the workspace
- inspect tools, resources, prompts, and raw manifest JSON
- show `mcp-risk-linter` findings in the Agent Studio view and hover provider
- open the same workspace in the desktop app via `auraone://agent-studio/open`
- host the compose surface in a webview when the desktop app is not installed

Marketplace publication requires the AuraOne publisher account and Microsoft
Marketplace review. The exact review blocker is tracked in
`docs/release/agent-studio-open-registry-blockers-2026-05-13.md`.
