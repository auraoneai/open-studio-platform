# Changelog

## 0.4.0 - 2026-07-12

- Added the OSS-safe `@auraone/proofline-oss` token and component foundation.
- Released `@auraone/platform-contracts` 0.3.0 with the shared Proofline
  contract consumed by the coordinated OSS surfaces.
- Released `@auraone/aura-ide-kit` 0.2.0 with light-first semantic Proofline styling while
  retaining compatibility APIs for the flagship Studio release train.
- Added public-asset verification for private fonts, remote fonts, and legacy
  global glass behavior.

All notable changes to the AuraOne Open Studio Platform are documented here.

## 0.1.0 - 2026-05-13

- Bootstrapped the Open Studio Platform repository with governance, schemas, Rust crates, Tauri template, AuraGlass IDE kit, docs template, signing wrappers, Worker service templates, installer distribution scaffolds, and security automation.
- Added v0.2 Robotics extension crates and v0.3 Agent extension crates with compile-time tests.
- Added verification artifacts and blocker tracking for external vendor gates.
