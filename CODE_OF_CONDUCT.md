# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

Participants are expected to be respectful, constructive, and focused on the work. Unacceptable behavior includes harassment, discriminatory language, threats, sustained disruption, and publication of another person's private information.

Report conduct issues to `opensource-conduct@auraone.ai`. Security issues must be reported through `security@auraone.ai` instead.
